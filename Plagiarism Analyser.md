Copy-Catcher | Plagiarism Analyzer

Copy-Catcher is a simple and interactive plagiarism detection tool built using **HTML, CSS, and JavaScript**.  
It allows users to paste text and analyze its similarity using a simulated plagiarism-checking algorithm.

Features:
-  Paste any text for analysis  
-  Instant similarity score using a mock algorithm  
-  Clean UI with smooth design  
-  Responsive and lightweight  
-  Beginner-friendly front-end project
